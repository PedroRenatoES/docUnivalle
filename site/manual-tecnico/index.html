    1. Instalación Base

    1 Requisitos del Sistema

    Hardware Mínimo

    • Procesador: CPU de 64 bits, multi-core
    • RAM: 8GB (recomendado 16GB)
    • Espacio en disco:
    • Android Studio: 8GB
    • Flutter SDK: 1.5GB
    • Android SDK y herramientas: 10GB
    • Emulador: 10GB por dispositivo virtual

    Software Base

    • Sistema Operativo:
    • Windows 10 versión 1903 o superior
    • Linux: Ubuntu 20.04 o superior
    • PowerShell 5.0 o superior (Windows)
    • Google Chrome (última versión)

    2 Instalación de Flutter

    2.1 Descargar Flutter SDK

    1. Visitar: https://flutter.dev/docs/get-started/install
    2. Descargar el archivo ZIP más reciente
    3. Crear directorio para Flutter
    4. Extraer el ZIP en el directorio creado

    2.2 . Configurar Variables de Entorno

    • Windows
    • Buscar "Variables de entorno" en el menú inicio
    • Clic en "Variables de entorno..."
    • En "Variables del sistema", seleccionar "Path"
    • Clic en "Nuevo"
    • Agregar: C:\src\flutter\bin

    2.3. Verificar Instalación

    3. Instalación de Node.js

    3.1. Instalación Base

    1. Visitar: https://nodejs.org
    2. Descargar versión LTS (recomendado 20.x.x)
    3. Ejecutar instalador:
      • Aceptar términos de licencia
      • Seleccionar directorio de instalación
      • Marcar "Automatically install necessary tools"
